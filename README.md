# Seesaw

Welcome to my playground. As you might expect from the name, you'll find various
random experiments in this repo, ranging from fake replicas of learning
management software to implementations of end-to-end encrypted messaging
algorithms.

## Contents

[X3DH](x3dh/): a rudimentary implementation of the X3DH exchange protocol in
Python

[Double Ratchet](dblratchet/): a rudimentary implementation of the Double
Ratchet message encryption protocol in Python

[Rust-like Errors in Go](go_errors/): an exploration of errors in Go and how we
could make them better (or worse, depending on your point of view)

[Multi-language Go Projects](multilang/): a brief exploration of running C or
Assembly as part of a Go program
